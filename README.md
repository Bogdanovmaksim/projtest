# Менеджер задач

Консольный проект на C++20. Пользователь может добавлять задачи, задавать описание, дедлайн, категорию и важность, сохранять данные в файл, смотреть список, фильтровать, сортировать по дедлайну, отмечать задачи выполненными, удалять, редактировать и архивировать завершенные задачи.

## Сборка

```bash
cmake -S . -B build
cmake --build build
./build/task_manager_app
```

## Тесты

```bash
ctest --test-dir build --output-on-failure
```

Тесты разделены по участникам:
- `tests/test_task_core.cpp` — даты, важность, статус, сериализация, фильтрация, сортировка.
- `tests/test_task_manager.cpp` — добавление, изменение, удаление, сохранение, загрузка, архив.

Тесты оформлены в стиле Практики 6: используется `doctest.h`, `TEST_CASE`, `CHECK_EQ` и `CHECK_THROWS_AS`, запуск подключен к CTest.

## Документация

В коде есть Doxygen-комментарии. HTML-документацию можно собрать командой:

```bash
doxygen Doxyfile
```

## Использование

При запуске можно указать файл задач и файл архива. Если нажать Enter, будут использованы `tasks.txt` и `archive.txt`.

Важность вводится по-русски:
- `низкая`
- `средняя`
- `высокая`

Статус при редактировании:
- `активна`
- `выполнена`
